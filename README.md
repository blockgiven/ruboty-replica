# Ruboty::Replica

ruboty plugin for replicate itself.

![screenshot](screenshot.png)

## Support
heroku

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-replica'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-replica

## Usage

    @ruboty replica

or

    @ruboty replica to ruboty@example.com
